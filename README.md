## ⚡ Quick Start

1. **Clone the Repository**
   ```bash
   git clone https://github.com/h3nr1d3v/NekoShell.git
   cd NekoShell
   ```

2. **Install Dependencies**

    ```bash
    pip install -r requirements.txt
    ```

3. **Configure Environment**
Create a `.env` file with your tokens:

    ```bash
    DISCORD_TOKEN=your_discord_token
    OPENAI_API_KEY=your_openai_token
    SPOTIFY_CLIENT_ID=your_spotify_client_id
    SPOTIFY_CLIENT_SECRET=your_spotify_client_secret
    ```

4. **Run the Bot**

    ```bash
    python main.py
    ```

You can view full installation guide [INSTALLATION](INSTALLATION.md)

## ⌨️ Essential Commands

| Category | Command | Description |
|----------|---------|-------------|
| **General** | `!help` | Shows help menu |
| | `!info` | Server information |
| | `!ping` | Check bot latency |
| | `!botinfo` | Bot statistics |
| **Moderation** | `!warn` | Warn a user |
| | `!ban` | Ban a user |
| | `!kick` | Kick a user |
| | `!mute` | Mute a user |
| **Music** | `!play` | Play a song |
| | `!queue` | Show queue |
| | `!skip` | Skip current song |
| | `!stop` | Stop playback |
| **Leveling** | `!rank` | Show your rank |
| | `!leaderboard` | Server leaderboard |
| | `!xp` | Check your XP |
| | `!rewards` | View level rewards |
| **Fun** | `!poll` | Create a poll |
| | `!translate` | Translate text |
| | `!8ball` | Ask the magic 8ball |
| | `!meme` | Get a random meme |
